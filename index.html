🔗 Tích hợp Google Sheets (Realtime Sync) 🔒 Khóa

🔒
Khu vực dành cho Admin
Nhập mật khẩu admin để truy cập cài đặt Google Sheets
❌ Mật khẩu không đúng. Vui lòng thử lại.
📊 Kết nối Google Sheets để lưu dữ liệu realtime. Mọi thao tác nhập/xuất sẽ tự động đồng bộ lên Google Sheets.
Lấy từ URL: docs.google.com/spreadsheets/d/[ID này]/edit
URL sau khi Deploy Apps Script → Web App
⚫ Chưa kết nối
—
1. Tạo Google Spreadsheet mới
2. Vào Extensions → Apps Script
3. Dán code Apps Script (xem bên dưới)
4. Deploy → Web App → Anyone can access
5. Copy URL và dán vào ô trên
📋 Google Apps Script Code
Copy toàn bộ code này vào Google Apps Script Editor → Deploy → Web App → Execute as: Me → Who has access: Anyone
